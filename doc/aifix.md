# Base78 类使用指南
- Base78 是大多数控制器类的基类，包含了许多通用方法和属性
- 在实现新功能或修改现有代码时，应首先查看 Base78 类的源代码
- Base78 类导入了许多核心服务和工具，包括：
  - DatabaseService
  - CacheService
  - AuthService
  - Config
  - UpInfo (从 'koa78-upinfo' 导入)
- 使用这些服务时，应参考 Base78 类中的使用方式
- UpInfo 是一个重要的工具类，用于处理请求参数和设置响应数据
- 在编写新的控制器或服务时，应该遵循 Base78 类的模式和约定

# 重要提示
- 在使用任何服务或工具类之前，务必检查它们在 Base78 类中的导入和使用方式
- 不要假设任何服务或工具类的存在或其方法，除非在 Base78 或其他相关类中明确看到
- 如果需要使用 UpInfo 或其他工具类，应该按照 Base78 类的方式导入：
  import UpInfo from 'koa78-upinfo';
- 在实现新功能时，应该先查看 Base78 类中是否已有类似的实现，以保持代码的一致性
